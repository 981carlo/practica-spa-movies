# Práctica SPA Movies

Repositorio de la práctica de **Desarrollo front con frameworks I**, en el que se desarrolla una aplicación de películas en dos versiones:

- **SPA con React + React Router DOM**
- **SSR con Next.js + App Router**

La aplicación permite:

- mostrar una lista de películas populares obtenidas desde la API de **TMDB**
- buscar películas por título
- ver el detalle de una película
- navegar entre páginas o secciones de la aplicación
